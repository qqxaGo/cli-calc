# CLI-Calc & Guess Game 🚀

Простое консольное приложение на **Go**, объединяющее калькулятор и игру "Угадай число". Проект создан для демонстрации работы с базовым синтаксисом языка.

## 🛠 Использованные начальные темы
В проекте использованы только базоывые начальные темы Go:
**go mod, переменные, const, базовые типы, fmt, if/else, switch**
---

## 📋 Функционал

### 1. Калькулятор (`Calc`)
Поддерживает ввод выражений в формате `число оператор число` (например, `10 + 5`).
* **Операции:** `+`, `-`, `*`, `/`, `%` (остаток), `^` (степень).
* **Особенности:** Защита от деления на ноль и форматирование результата до 2 знаков после запятой.
* **Выход:** Введите `0 0 0`.

### 2. Игра "Угадай число" (`Guess`)
* Программа генерирует случайное число от 0 до 100.
* Пользователь вводит варианты, получая подсказки "Больше" или "Меньше", пока не найдет верное число.
